## 使用
```
{ name:"用戸名", 
  text:"正文内容", 
  time:"YYYY-MM-DD", 
  address:"所在处/Unknown",
},
```

## 投稿须知
- 复制上述代码编编并回执我们的邮箱投稿
- 请修改代码中 "用戸名" "正文内容" "YYYY-MM-DD" "所在处"字段
- YYYY-MM-DD为年月日 例如2008-01-02
- 所在处仅辅助作用 如不愿提供保留Unknown即可
- 文字不支援换行
- 文字不支援超链接 图片等任何除文本外的媒体

- 您所投递的内容不得违反法规道德等
- 您若投递则视为您已悉知并同意上述内容
